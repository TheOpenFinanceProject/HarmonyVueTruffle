# Harmony Vue Vuetify VueX Router Open-zeppellin-contracts

Simply compile the contracts, serve the page, sign in, and use

-One Wallet Support
-Truffle Developement
-Vue/Vuetify
-Vuex/Router
-Linter
-Harmony Contract Interaction

For Web developpers.

## Project setup
```
npm install
```
```
add keys to .env file
```
```
truffle compile
```
